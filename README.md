# Behavioral Telemetry Framework for AI-Assisted Programming

This repository contains the dataset and code used in the paper:

**"Behavioral Telemetry Reveals Cognitive Archetypes in AI-Assisted Problem Solving"**

## Contents

* `synthetic_telemetry_51_signals.csv` — Synthetic dataset with 51 behavioral features
* `generate_synthetic_telemetry.py` — Script for dataset generation
* Analysis scripts / notebooks

## Description

The dataset simulates behavioral patterns in AI-assisted programming sessions, capturing:

* Coding activity
* Typing dynamics
* Code evolution
* AI interaction behavior
* Temporal workflow patterns

## Usage

Run the Python script to generate synthetic data or use the provided dataset for analysis.

## Notes

* The dataset is fully synthetic
* No human subjects or personal data are involved
